AlgMe

Make learning algs on the web easier!

About

AlgMe is a Javascript/jQuery bookmarklet that allows you to play highlighted Rubik's Cube algorithms on a website. It uses the alg.cubing.net visualizer.

How to Use

  1. Drag the link above into your bookmarks bar.
  2. Go to a site with some cube algorithms that you need to learn.
  3. Click the bookmarklet.
  4. Adjust the settings.
  5. Highlight an algorithm. You can test here:
    M2 E2 S2
  6. Repeat step 5 if needed, and enjoy!
