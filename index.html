AlgMe

Make learning algs on the web easier!

About

AlgMe is a Javascript/jQuery bookmarklet that allows you to play highlighted Rubik's Cube algorithms on a website. It uses the alg.cubing.net cube visualizer and jquery.selection plugin.

How to Use

  1. Drag the link above that says "AlgMe" into your bookmarks bar.
  2. Go to a site with some cube algorithms that you need to learn.
  3. Click the bookmarklet.
  4. Adjust the settings if needed.
  5. Highlight an algorithm and play it. You can test here:
  6. Repeat step 5 if needed, and enjoy!
